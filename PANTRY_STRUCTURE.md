# Melius Operarius Pantry Structure

The Melius Operarius Engine reads all its instructions from a Pantry bucket. The Pantry ID should be stored in the `PANTRY_ID` environment variable.

## 1. Main Instructions Basket (`melius_instructions`)
This is the primary bucket where the AI gets its directions.

```json
{
  "theme": {
    "description": "User's theme description (e.g., 'Modern dark mode with neon blue accents')",
    "primary_color": "#00d4ff",
    "secondary_color": "#0055ff",
    "background_color": "#121212",
    "text_color": "#ffffff"
  },
  "special_mentions": "Additional specific instructions for the AI (can be blank)",
  "pages": [
    {
      "id": "home",
      "title": "Welcome Page",
      "theme": null,
      "content": "This is our homepage. Here is a {form}Contact Us{/form} and a {countdown}2026-12-31{/countdown}.",
      "strict_text": "This text must remain exactly as written.",
      "offer_details": "New Year Special: 20% off!"
    }
  ],
  "special_tags_config": {
    "form": "AI creates a new bucket in same pantry for each form",
    "countdown": "Target date for the timer",
    "live_time": "Displays current time",
    "sales_banner": "Displays a sales message",
    "discount_banner": "Displays a discount message",
    "product_banner": "Format: Name|ImageURL|Price|ContactInfo",
    "image": "Direct link to image",
    "video": "Direct link to video"
  }
}
```

### Key Fields:
- **theme**: AI modifies colors to apply this theme. If hex codes are null, AI decides based on description.
- **pages**: List of pages to manage/create.
  - **content**: AI can improve this text (e.g., formatting into 2 columns if comparisons are found).
  - **strict_text**: AI MUST use this text exactly as provided, no changes allowed.
  - **title**: If null, AI decides.
  - **theme**: If null, uses main app theme.
- **special_tags**: Attributes like `{form}`, `{countdown}`, `{live_time}`, etc., used within `content`.

## 2. Forms Registry Basket (`melius_forms`)
Stores the names of all buckets created for form submissions.

```json
{
  "forms": [
    {
      "form_id": "contact_us_home",
      "bucket_name": "form_contact_us_home_data",
      "created_at": "2026-02-13T10:00:00Z"
    }
  ]
}
```

## 3. Form Data Baskets (Dynamic)
Each form created by the AI will have its own basket to store submissions. The name of this basket is generated by the AI and saved in the `melius_forms` registry.
